public ScopedLockInterface acquire(string $resource, integer|float|null $ttl = null, integer|float $timeout = INF)

Attempt to acquire a lock and throw an exception if acquisition is unsuccessful.

This is a convenience method analogous to calling acquire() on the lock returned by a call to $this->create($resource).

Parameters

string $resource The resource to lock.
integer|float|null $ttl How long the lock should persist, in seconds, or null to use the default.
integer|float $timeout How long to wait for lock acquisition, in seconds.

Return Value

ScopedLockInterface An acquired lock for the given resource.

Exceptions

LockException if the lock can not be acquired.
